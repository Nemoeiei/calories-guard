## 4. 🔬 Test Process (ขั้นตอนการทดสอบ 14 วัน)

### ก่อนเริ่ม
- [ ] วัดน้ำหนักเริ่มต้น: **70 กก.**
- [ ] ติดตั้งแอปและสมัครสมาชิก
- [ ] กรอกโปรไฟล์ครบถ้วน
- [ ] ตรวจสอบว่า target_calories = **1,258 kcal**

### วันที่ 1–7 (สัปดาห์แรก)
- [ ] บันทึกอาหารครบ 3 มื้อ/วัน ทุกวัน
- [ ] ตรวจสอบแคลอรี่สะสมในหน้า Home
- [ ] ดูอาหารแนะนำโปรตีนอย่างน้อย 1 ครั้ง
- [ ] บันทึกน้ำดื่มทุกวัน

### วันที่ 7 (ครึ่งทาง)
- [ ] อัปเดตน้ำหนัก ณ วันที่ 7
- [ ] ตรวจสอบ % ความสำเร็จว่าเพิ่มขึ้นหรือไม่
- [ ] ดูกราฟ Progress สัปดาห์แรก

### วันที่ 8–14 (สัปดาห์ที่สอง)
- [ ] บันทึกอาหารต่อเนื่อง
- [ ] ลองเพิ่มเมนูอาหารที่ไม่มีในระบบ
- [ ] ตรวจสอบการแจ้งเตือน (calorie warning / alert)

### วันที่ 14 (สิ้นสุด)
- [ ] วัดน้ำหนักสุดท้าย
- [ ] ดูสรุปภาพรวม 2 สัปดาห์
- [ ] บันทึกผลการทดสอบ

### UC-08 — ดูเปอร์เซ็นต์ความสำเร็จ

| สถานการณ์ | น้ำหนัก | % สำเร็จ | Expected Display |
|---|---|---|---|
| เริ่มต้น | 70 kg | 0% | "เพิ่งเริ่มต้น" |
| สัปดาห์ที่ 1 | 69.5 kg | ~17% | Progress bar เพิ่มขึ้น |
| สัปดาห์ที่ 2 | 68.8 kg | ~40% | Progress bar เพิ่มขึ้น |
| เสร็จสิ้น | 67 kg | 100% | 🎉 "ถึงเป้าหมายแล้ว!" |

**สูตรคำนวณ % สำเร็จ:**
```
% = (น้ำหนักเริ่มต้น − น้ำหนักปัจจุบัน) / (น้ำหนักเริ่มต้น − น้ำหนักเป้าหมาย) × 100
= (70 − current) / (70 − 67) × 100
```

| # | ขั้นตอน | Expected Result | Pass/Fail |
|---|---|---|---|
| 8.1 | อัปเดตน้ำหนัก 69.5 kg (วันที่ 7) | % เพิ่มขึ้นจาก 0% | ☐ |
| 8.2 | ตรวจ Progress bar | แสดงเพิ่มขึ้น ไม่คงเดิม | ☐ |
| 8.3 | ตรวจ DB `weight_logs` | มี record ใหม่ recorded_date = วันที่ 7 | ☐ |
| 8.4 | % ไม่ควรเกิน 100% ก่อนถึงเป้า | cap ที่ 100% | ☐ |

## 10. 🗃️ SQL Queries สำหรับตรวจสอบผล

```sql
-- ตรวจสอบข้อมูล user ที่สร้างขึ้น
SELECT user_id, username, gender, height_cm, current_weight_kg,
       target_weight_kg, target_calories, goal_type
FROM cleangoal.users
WHERE username = 'somying_test';

-- ดูสรุปแคลอรี่รายวัน 14 วัน
SELECT date_record, total_calories_intake, total_protein,
       total_carbs, total_fat, water_glasses
FROM cleangoal.daily_summaries
WHERE user_id = [USER_ID]
ORDER BY date_record;

-- นับวันที่บันทึกครบ
SELECT COUNT(*) AS days_recorded
FROM cleangoal.daily_summaries
WHERE user_id = [USER_ID]
  AND total_calories_intake > 0;

-- ค่าเฉลี่ยแคลอรี่
SELECT ROUND(AVG(total_calories_intake), 0) AS avg_calories,
       ROUND(AVG(total_protein), 1) AS avg_protein
FROM cleangoal.daily_summaries
WHERE user_id = [USER_ID];

-- ตรวจสอบน้ำหนัก
SELECT recorded_date, weight_kg
FROM cleangoal.weight_logs
WHERE user_id = [USER_ID]
ORDER BY recorded_date;

-- คำนวณ % ความสำเร็จ
SELECT
  first_weight.weight_kg AS start_weight,
  last_weight.weight_kg AS current_weight,
  70 AS initial_weight,
  67 AS target_weight,
  ROUND(
    ((70 - last_weight.weight_kg) / (70 - 67)) * 100, 1
  ) AS success_pct
FROM
  (SELECT weight_kg FROM cleangoal.weight_logs WHERE user_id=[ID] ORDER BY recorded_date ASC LIMIT 1) first_weight,
  (SELECT weight_kg FROM cleangoal.weight_logs WHERE user_id=[ID] ORDER BY recorded_date DESC LIMIT 1) last_weight;
```
